Aplicativo desenvolvido para fins acadêmicos: Andrômeda – Joias Exclusivas, versão mobile da loja virtual.
O projeto oferece uma experiência intuitiva para navegação, visualização de produtos e utilização de funcionalidades
como catálogo de joias, pesquisa de produtos, formulário de contato e reprodução de vídeos, com foco em usabilidade,
e experiência do usuário.
